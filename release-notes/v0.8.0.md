### Added

- Status filter — filter resources by action (create, destroy, update, etc.)
- Sort — sort resource list by default, action, address, or type
- Sort picker includes hints for each option
- Esc key clears filters from main view or filter picker

### Usage

In the TUI (after `terraprism plan` or `terraprism apply`):

1. **Filter by status** — open the filter picker and choose an action (e.g. `create`, `destroy`)
2. **Sort** — open the sort picker and choose default, action, address, or type
3. **Clear filters** — press Esc to reset filters or exit the picker
