### Added

- `TERRAPRISM_TOFU` — set to 1, true, or yes to use OpenTofu instead of Terraform
- `TERRAPRISM_THEME` — set to "light" or "dark" to force color scheme
- CLI flags override env vars when both are specified

### Usage

```bash
# Use OpenTofu instead of Terraform (no --tofu flag needed)
TERRAPRISM_TOFU=1 terraprism plan

# Force dark theme
TERRAPRISM_THEME=dark terraprism apply

# Add to your shell profile for persistence
export TERRAPRISM_TOFU=1
export TERRAPRISM_THEME=dark
```
