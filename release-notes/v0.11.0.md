### Added

- **Live search filter** — resources filter as you type in the search bar (`/`)
  - Fuzzy matching — query characters match in order (e.g. `lmbda` matches `lambda`)
  - Multi-term search — space-separated terms, all must match (fzf-style)
  - Arrow keys (up/down) scroll the filtered list while the search bar is active
  - Esc clears search and returns to the full list; j/k are typed into the search box
- **Fail-safe userdata encoding pipeline** — improved handling of base64/gzip/hex userdata in plan diffs
- `terraprism upgrade` — upgrade to the latest release with a single command
- Version check — `terraprism version` shows an update hint when a newer version is available
- TUI footer nudge — when a newer version exists, a timely check (7-day cache) displays an update prompt at the bottom of the screen
- Curl fallback — if self-update fails (e.g. read-only install location), the command prints the manual upgrade instruction
- `TERRAPRISM_SKIP_UPDATE_CHECK` — set to 1, true, or yes to skip all update checks
- `TERRAPRISM_UPDATE_CHECK_INTERVAL` — days between TUI update checks (default: 7)

### Usage

```bash
# In TUI: press / to search, type to filter live, arrow keys to scroll results
terraprism plan

# Check for updates (shown when running version)
terraprism version

# Upgrade to the latest release
terraprism upgrade

# If self-update fails (read-only install), re-run the install script
curl -sSfL https://raw.githubusercontent.com/CaptShanks/terraprism/main/install.sh | sh
```

### Upgrade from v0.10.0

Run `terraprism upgrade` or re-run the install script to get the latest binary.
